# CIMSO Hackathon Golf Booking

A Flutter mobile app for booking golf tee times, built for the CIMSO Hackathon 2025.

## 🎨 Design

Figma design file: [ICT425_Cimso_Hackathon_2025](https://www.figma.com/design/F8BFDipVxIq5By2yXSD15t/ICT425_Cimso_Hackathon_2025?node-id=60-13&t=1SuRvrNwad70ESZf-1)

## ✨ Features

- User authentication (Register, Login, OTP verification, password reset)
- Browse golf courses and book tee times
- Booking confirmation and booking history
- User profile management
- Location-based course search (map integration)

## 📱 Screens

- Login / Register / OTP Verification
- Forgot / Change Password
- Course Listing
- Book Tee Time
- Booking Complete / Check Booking
- Profile Edit

## 🛠️ Tech Stack

- **Flutter** / Dart
- `url_launcher` — external links
- `image_picker` — profile photos
- `flutter_map` + `latlong2` — course location & maps
- `country_code_picker` — phone number input
